## Verdict: Real

I read through [this guide](https://gameverifying.com/wiki/cart-based-systems/nds) and concluded that the cart I now own is a legitimate copy. Here's why:

1. The label on the front of the cart appears neatly centred
2. The PCB Batch Identifier "DA" (visible on the part of the chip that is exposed) is not yet known to be used by fake carts
3. The inked code on the back of the cart (which is barely visible) has 9 digits as NTR-005 carts should, and it matches the "ADAF" code sequence on the front label. The code itself "ADAFN5J08", if I'm reading it correctly, matches verified FRA carts
4. The PCB variation (the last segment of code visible on the chip) "C03-10" matches verified FRA carts
